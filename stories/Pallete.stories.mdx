# Color Pallettes

---

The theme color palette is used to define the color of the primary elements of the application.



```typescript
/* cu-ng-design-library/src/lib/styles.scss */
// Create the theme object. A theme consists of configurations for individual
// theming systems such as "color" or "typography".
$styleguide-theme: mat.define-light-theme((
  color: (
    primary: $styles-primary-color,
    accent: $styles-accent-color,
    warn: $styleguide-warn,
  )
));
```

also see `/* cu-ng-design-library/src/lib/theme/_palette.scss */` for color pallettes definitions.
